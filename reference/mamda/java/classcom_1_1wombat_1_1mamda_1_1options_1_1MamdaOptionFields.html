Set the dictionary for common options fields.

Maintains a cache of MamaFieldDescriptors for common options related fields. The properties parameter allows users of the API to map the common dictionary names to something else if they are being published under different names.

Parameters
dictionaryA reference to a valid MamaDictionary
propertiesA Properties object containing field mappings. (See MamdaFields for more information)
