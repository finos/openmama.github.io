Detailed Description

Utility cache of

MamaFieldDescriptors which are used internally by the API when accessing imbalance related fields from update messages. This class should be initialized prior to using the MamdaOrderImbalanceListener by calling setDictionary() with a valid dictionary object which contains imbalance related fields.
