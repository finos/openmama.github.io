Detailed Description

A file-based symbol mapper

Use MamaSymbolMapFile load() to open the file. If the file exists, its contents should be a two columns of data, with the columns separated by white space. A matching symbol of the left column is mapped to the symbol in the right column. When performing the mapping, if the symbol does not match anything in the file (or the file cannot be found), then the original symbol is used (i.e. no mapping)
