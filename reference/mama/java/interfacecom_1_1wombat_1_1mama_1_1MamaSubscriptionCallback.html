Detailed Description

The message callback interface.

Callers provide an object implementing this interface on creating a MamaSubscription.

§ onCreate()

void com.wombat.mama.MamaSubscriptionCallback.onCreate ( MamaSubscription  subscription)

Method invoked when subscription creation is complete.

Since subscriptions are created asynchronous by throttle, this callback provides the subscription instance after the throttle processes the creation request. In the case where a subscription is created on a queue other than the default it is possible for onMsg calls to be processed to be called before the onCreate callback is processed.

subscriptionThe subscription.

Invoked if an error occurs during subscription creation or if the subscription receives a message for an unentitled subject.

If the status is MamaStatus.NOT_ENTITTLED the subject parameter is the specific unentitled subject. If the subscription subject contains wildcards, the subscription may still receive messages for other entitled subjects.
